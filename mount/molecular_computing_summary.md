## The Breakthrough

**One molecule** can function as:
1. Memory (via redox states)
2. Logic gate (via electron coupling)
3. Learning synapse (via ion migration)

This is NOT "better silicon" — it's a **different computational paradigm**.

---

## How It Works

### Three states, three functions

**Redox states** (fast, electrons):
- Ru²⁺ (reduced) — low conductance
- Ru³⁺ (neutral) — medium conductance
- Ru⁴⁺ (oxidized) — high conductance

→ Conductance = memory bit (0 or 1)

**Electron coupling** (picoseconds):
- Multiple devices connected
- Currents combine (series/parallel)

→ Logic operations (AND, OR, XOR)

**Ion migration** (milliseconds):
- Counterions drift in molecular matrix
- Changes threshold for switching
- Persists as "learned weight"

→ Synaptic plasticity (Hebbian learning)
