<style data-mw-deduplicate="TemplateStyles:r1320445320">.mw-parser-output .hatnote{font-style:italic}.mw-parser-output div.hatnote{padding-left:1.6em;margin-bottom:0.5em}.mw-parser-output .hatnote i{font-style:normal}.mw-parser-output .hatnote+link+.hatnote{margin-top:-0.5em}@media print{body.ns-0 .mw-parser-output .hatnote{display:none!important}}</style><div role="note" class="hatnote navigation-not-searchable">This article is about the theory of source coding in data compression. For the term in computer programming, see <a href="/wiki/Source_code" title="Source code">Source code</a>.</div>
<p>In <a href="/wiki/Information_theory" title="Information theory">information theory</a>, <b>Shannon's source coding theorem</b> (or <b>noiseless coding theorem</b>) establishes the statistical limits to possible <a href="/wiki/Data_compression" title="Data compression">data compression</a> for data whose source is an <a href="/wiki/Independent_identically-distributed_random_variables" class="mw-redirect" title="Independent identically-distributed random variables">independent identically-distributed random variable</a>, and the operational meaning of the <a href="/wiki/Shannon_entropy" class="mw-redirect" title="Shannon entropy">Shannon entropy</a>.
</p><p>Named after <a href="/wiki/Claude_Shannon" title="Claude Shannon">Claude Shannon</a>, the <b>source coding theorem</b> shows that, in the limit, as the length of a stream of <a href="/wiki/Independent_and_identically_distributed_random_variables" title="Independent and identically distributed random variables">independent and identically-distributed random variable (i.i.d.)</a> data tends to infinity, it is impossible to compress such data such that the code rate (average number of bits per symbol) is less than the Shannon entropy of the source, without it being virtually certain that information will be lost. However it is possible to get the code rate arbitrarily close to the Shannon entropy, with negligible probability of loss.
</p><p>The <b>source coding theorem for symbol codes</b> places an upper and a lower bound on the minimal possible expected length of codewords as a function of the <a href="/wiki/Entropy_(information_theory)" title="Entropy (information theory)">entropy</a> of the input word (which is viewed as a <a href="/wiki/Random_variable" title="Random variable">random variable</a>) and of the size of the target alphabet.
</p><p>Note that, for data that exhibits more dependencies (whose source is not an i.i.d. random variable), the <a href="/wiki/Kolmogorov_complexity" title="Kolmogorov complexity">Kolmogorov complexity</a>, which quantifies the minimal description length of an object, is more suitable to describe the limits of data compression. Shannon entropy takes into account only frequency regularities while Kolmogorov complexity takes into account all algorithmic regularities, so in general the latter is smaller. On the other hand, if an object is generated by a random process in such a way that it has only frequency regularities, entropy is close to complexity with high probability (Shen et al. 2017).<sup id="cite&#95;ref-Shen2017&#95;1-0" class="reference"><a href="#cite_note-Shen2017-1"><span class="cite-bracket">&#91;</span>1<span class="cite-bracket">&#93;</span></a></sup>
</p>
<meta property="mw:PageProp/toc" />
<div class="mw-heading mw-heading2"><h2 id="Statements">Statements</h2><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Shannon%27s_source_coding_theorem&amp;action=edit&amp;section=1" title="Edit section: Statements"><span>edit</span></a><span class="mw-editsection-bracket">]</span></span></div>
<p><i>Source coding</i> is a mapping from (a sequence of) symbols from an information <a href="/wiki/Information_theory#Source_theory" title="Information theory">source</a> to a sequence of alphabet symbols (usually bits) such that the source symbols can be exactly recovered from the alphabet symbols (lossless source coding) or recovered within some distortion (lossy source coding). This is one approach to <a href="/wiki/Data_compression" title="Data compression">data compression</a>.
</p>
<div class="mw-heading mw-heading3"><h3 id="Source_coding_theorem">Source coding theorem</h3><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Shannon%27s_source_coding_theorem&amp;action=edit&amp;section=2" title="Edit section: Source coding theorem"><span>edit</span></a><span class="mw-editsection-bracket">]</span></span></div>
<p>In information theory, the source coding theorem (Shannon 1948)<sup id="cite&#95;ref-Shannon&#95;2-0" class="reference"><a href="#cite_note-Shannon-2"><span class="cite-bracket">&#91;</span>2<span class="cite-bracket">&#93;</span></a></sup> informally states that (MacKay 2003, pg. 81,<sup id="cite&#95;ref-MacKay&#95;3-0" class="reference"><a href="#cite_note-MacKay-3"><span class="cite-bracket">&#91;</span>3<span class="cite-bracket">&#93;</span></a></sup> Cover 2006, Chapter 5<sup id="cite&#95;ref-Cover&#95;4-0" class="reference"><a href="#cite_note-Cover-4"><span class="cite-bracket">&#91;</span>4<span class="cite-bracket">&#93;</span></a></sup>):
</p>
<blockquote><p><span class="texhtml mvar" style="font-style:italic;">N</span> <a href="/wiki/Independent_and_identically_distributed_random_variables" title="Independent and identically distributed random variables">i.i.d.</a> random variables each with entropy <span class="texhtml"><i>H</i>(<i>X</i>)</span> can be compressed into more than <span class="texhtml"><i>N&#8201;H</i>(<i>X</i>)</span> <a href="/wiki/Bit" title="Bit">bits</a> with negligible risk of information loss, as <span class="texhtml"><i>N</i> → ∞</span>; but conversely, if they are compressed into fewer than <span class="texhtml"><i>N&#8201;H</i>(<i>X</i>)</span> bits, it is virtually certain that information will be lost.</p></blockquote><p> The coded sequence of length <span class="mwe-math-element mwe-math-element-inline"><span class="mwe-math-mathml-inline mwe-math-mathml-a11y" style="display: none;"><math xmlns="http://www.w3.org/1998/Math/MathML"  alttext="{\displaystyle NH(X)}">

<p>Given <span class="texhtml mvar" style="font-style:italic;">X</span> is an <a href="/wiki/Independent_identically-distributed_random_variables" class="mw-redirect" title="Independent identically-distributed random variables">i.i.d.</a> source, its <a href="/wiki/Time_series" title="Time series">time series</a> <span class="texhtml"><i>X</i><sub>1</sub>, ..., <i>X<sub>n</sub></i></span> is i.i.d. with <a href="/wiki/Entropy_(information_theory)" title="Entropy (information theory)">entropy</a> <span class="texhtml"><i>H</i>(<i>X</i>)</span> in the discrete-valued case and <a href="/wiki/Differential_entropy" title="Differential entropy">differential entropy</a> in the continuous-valued case. </p><p><i>Proof of converse</i>: the converse is proved by showing that any set of size smaller than <span class="texhtml"><i>A</i><span class="nowrap"><span style="display:inline-block;margin-bottom:-0.3em;vertical-align:-0.4em;line-height:1.2em;font-size:80%;text-align:left"><sup style="font-size:inherit;line-height:inherit;vertical-align:baseline"><i>ε</i></sup><br /><sub style="font-size:inherit;line-height:inherit;vertical-align:baseline"><i>n</i></sub></span></span></span> (in the sense of exponent) would cover a set of probability bounded away from <span class="texhtml">1</span>.
